## 🛠️ 环境搭建

### 1.1 WSL环境检查

```bash
# 检查 WSL 版本
wsl -l -v

# 进入 WSL
wsl -u jules
```

### 1.2 启动 Hadoop

```bash
# 启动 HDFS
/usr/local/hadoop/sbin/start-dfs.sh

# 启动 YARN
/usr/local/hadoop/sbin/start-yarn.sh

# 检查进程
jps

# 应该看到：
# NameNode
# DataNode
# SecondaryNameNode
# ResourceManager
# NodeManager
```

### 1.3 启动 MySQL

```bash
# Windows 中检查 MySQL
# 查看 MySQL 服务状态
# net start mysql
```

### 1.4 初始化 Hive 数据库

```bash
# 进入 gmall-dw 目录
cd /mnt/d/s/作业/gmall-dw

# 使用 hive.sh 脚本初始化数据库
```

## 📂 ODS层加载

### 3.1 创建 ODS 层表

```sql
-- 执行 ODS 层建表脚本
hive -f hive/create_ods.sql
```

### 3.2 加载 ODS 分区

```sql
-- 添加订单表分区
ALTER TABLE gmall_ods.ods_order_info
ADD PARTITION(dt='2020-06-14')
LOCATION '/origin_data/gmall/db/order_info_inc/2020-06-14';

-- 添加订单明细表分区
ALTER TABLE gmall_ods.ods_order_detail
ADD PARTITION(dt='2020-06-14')
LOCATION '/origin_data/gmall/db/order_detail_inc/2020-06-14';

-- 添加用户表分区
ALTER TABLE gmall_ods.ods_user_info
ADD PARTITION(dt='2020-06-14')
LOCATION '/origin_data/gmall/db/user_info_full/2020-06-14';
```

### 3.3 验证 ODS 数据

```sql
-- 检查数据量
SELECT COUNT(*) FROM gmall_ods.ods_order_info WHERE dt='2020-06-14';
SELECT COUNT(*) FROM gmall_ods.ods_order_detail WHERE dt='2020-06-14';
SELECT COUNT(*) FROM gmall_ods.ods_user_info WHERE dt='2020-06-14';

-- 查看样例数据
SELECT * FROM gmall_ods.ods_order_info WHERE dt='2020-06-14' LIMIT 10;
```

## 📋 常用 Hive 命令

### 查看数据库

```sql
SHOW DATABASES;
USE gmall_ods;
SHOW TABLES;
```

### 查看表结构

```sql
DESCRIBE gmall_ods.ods_order_info;
DESCRIBE FORMATTED gmall_ods.ods_order_info;
```

### 查看分区

```sql
SHOW PARTITIONS gmall_ods.ods_order_info;
```

### 添加/删除分区

```sql
-- 添加分区
ALTER TABLE table_name ADD PARTITION(dt='2020-06-14') LOCATION '/path/to/data';

-- 删除分区
ALTER TABLE table_name DROP PARTITION(dt='2020-06-14');
```

## ⚠️ 常见问题

### HDFS目录不存在

```bash
# 创建目录
hdfs dfs -mkdir -p /warehouse/gmall/ods
hdfs dfs -chmod -R 755 /warehouse/gmall
```

### Hive 分区修复

```sql
MSCK REPAIR TABLE table_name;
```

### DataX 内存不足

修改 DataX 启动脚本，调整 JVM 内存参数。
