### 数据仓库分层架构 (5层)

```
MySQL(业务库) ──Python ETL──> HDFS ──Hive External Tables──> ODS(原始数据层)
                                                                   │
                                                    ┌──────────────┼──────────────┐
                                                    ▼              ▼              ▼
                                                 DIM(维度层)   DWD(明细层)    DWS(汇总层)
                                                                                   │
                                                                                   ▼
                                                                              ADS(应用层)
                                                                                   │
                                                                              Superset(报表)
```

## 四、技术亮点

### 1. 数据脱敏
DWD 层对敏感字段进行数据脱敏处理（商品名），展示数据安全治理能力。

### 2. 拉链表设计
DIM 层用户维度采用 SCD Type 2 缓慢变化维设计（项目 SQL 脚本支持）。

### 3. 全链路数据流
实现了 `MySQL → HDFS → ODS → DIM/DWD → DWS → ADS` 五层数据仓库完整链路。

### 4. Docker 容器化
全组件 Docker 容器化部署，包括 Hadoop HDFS、Hive、YARN、MySQL，具备生产级部署形态。

### 5. 指标可追溯
从 ADS 指标可逐层下钻追溯到 ODS 原始数据，满足数据血缘追溯要求。
