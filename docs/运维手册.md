## 三、性能优化

### 3.1 Hive 优化

```sql
-- 1. 启用向量化查询
SET hive.vectorized.execution.enabled=true;

-- 2. 开启 CBO 优化
SET hive.cbo.enable=true;

-- 3. 调整并行度
SET mapreduce.job.reduces=10;

-- 4. 启用压缩
SET hive.exec.compress.output=true;
SET mapreduce.output.fileoutputformat.compress.codec=org.apache.hadoop.io.compress.SnappyCodec;

-- 5. 小表广播
SET hive.auto.convert.join=true;
SET hive.mapjoin.smalltable.filesize=25000000;
```

### 3.2 数据倾斜处理

```sql
-- 1. 加盐处理
SELECT
    CONCAT(user_id, '_', FLOOR(RAND() * 10)) AS user_id_salt,
    COUNT(*)
FROM dwd_order_info
GROUP BY CONCAT(user_id, '_', FLOOR(RAND() * 10));

-- 2. 空值处理
SELECT
    CASE WHEN user_id IS NULL THEN CONCAT('NULL_', RAND()) ELSE user_id END,
    COUNT(*)
FROM dwd_order_info
GROUP BY CASE WHEN user_id IS NULL THEN CONCAT('NULL_', RAND()) ELSE user_id END;
```

### Q3: 数据重复怎么办？

```sql
-- 去重处理
INSERT OVERWRITE TABLE table_name PARTITION(dt='${biz_date}')
SELECT * FROM (
    SELECT *, ROW_NUMBER() OVER (PARTITION BY id ORDER BY etl_time DESC) AS rn
    FROM source_table
) t WHERE t.rn = 1;
```
