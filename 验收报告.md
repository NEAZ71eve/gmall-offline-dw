## 三、ETL 流水线验证

### 3.1 ETL 执行流程

```
MySQL(业务库)
  │  Python读取 → TSV导出
  ▼
HDFS ODS (原始数据)
  │  Python读取 → 维度提取
  ├──→ DIM (维度表)
  │     ├── dim_time: 366行
  │     ├── dim_sku: 6行
  │     └── dim_user: 8行
  │  Python读取 → 关联清洗
  ├──→ DWD (明细表)
  │     └── dwd_order_detail: 8行
  │  Python聚合
  ├──→ DWS (汇总表)
  │     └── dws_gmv_stats: 1行
  │  Python指标计算
  └──→ ADS (应用表)
        ├── ads_gmv_day: GMV=¥15,891
        ├── ads_user_retention: 留存率75%
        └── ads_sku_sales_rank: 5个SKU
```

### 3.2 数据血缘

```
ads_gmv_day.gmv = SUM(dwd_order_detail.price * dwd_order_detail.num)
                = SUM(ods_order_detail joined with ods_order_info)
                = MySQL(order_detail + order_info)
```
