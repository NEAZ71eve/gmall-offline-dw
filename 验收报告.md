## 快速启动指南

### 1. 启动基础服务
```bash
./start_all.sh          # MySQL + Hadoop
./start_kafka.sh        # ZooKeeper + Kafka + Maxwell
./start_flume.sh        # Flume采集
```

### 2. 执行ETL流程
```bash
# 方式1：Python脚本
python demo_etl_flow.py

# 方式2：DataX
./datax/run_datax.sh

# 方式3：DolphinScheduler
./dolphinscheduler/submit_workflow.sh
```

### 3. 查看数据
