## ⚠️ 注意事项

### Hive 与 Java 11 兼容性问题
Hive 3.1.3 与 Java 11 存在兼容性问题（ClassCastException）。解决方案：
1. 安装 Java 8 并配置 HIVE 使用 Java 8
2. 升级到 Hive 4.0（支持 Java 11）
3. 使用 Spark SQL 替代 Hive CLI
