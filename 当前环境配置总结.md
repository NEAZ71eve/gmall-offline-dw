## 四、数据流架构

### 4.1 当前数据流

```
┌─────────────┐
│   MySQL    │
│ (school.db) │
└──────┬──────┘
       │ pymysql
       ↓
┌─────────────────────────┐
│  mysql_to_kafka_loop.py │
│  (每10秒执行一次)        │
└──────────┬──────────────┘
           │ stdout (JSON)
           ↓
┌─────────────────────────┐
│    Flume Exec Source    │
└──────────┬──────────────┘
           │ Events
           ↓
┌─────────────────────────┐
│   Flume Memory Channel  │
└──────────┬──────────────┘
           │
           ↓
┌─────────────────────────┐
│    Flume Kafka Sink     │
│  (flume-mysql-topic)    │
└──────────┬──────────────┘
           │ Kafka Protocol
           ↓
┌─────────────────────────┐
│         Kafka          │
│  (localhost:9092)       │
│                        │
│ Topics:                │
│ - flume-mysql-topic    │
│ - flume-input-topic     │
│ - student-topic         │
└──────────┬──────────────┘
           │
           ├──→ Consumer (kafka_to_console_consumer.py)
           │
           └──→ kafka_to_hdfs.py → HDFS
```

### 4.2 数据存储位置

| 数据源 | 目的地 | 格式 |
|--------|--------|------|
| MySQL student表 | Kafka flume-mysql-topic | JSON |
| Kafka flume-input-topic | HDFS /fromkafka/ | Text |
