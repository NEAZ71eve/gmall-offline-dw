## 二、数据流向

```
业务系统 → DataX/Flume → HDFS → Hive ODS → Hive DIM → Hive DWD → Hive DWS → Hive ADS → 报表/大屏
```

## 三、技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 采集 | DataX | MySQL数据同步到HDFS |
| 采集 | Flume | 日志采集到HDFS |
| 存储 | HDFS | 分布式文件系统 |
| 计算 | Hive | 数据仓库 |
| 调度 | Azkaban | 任务调度 |
| 可视化 | Superset | 报表工具 |
| 可视化 | DataV | 数据大屏 |
