## 快速启动脚本

如果Kafka服务停止，可以使用以下命令重启：

```bash
cd /mnt/d/s/作业
chmod +x restart_kafka_full.sh
./restart_kafka_full.sh
```

等待约15秒后，重新创建topics：

```bash
cd /opt/kafka
bin/kafka-topics.sh --create --topic student-topic --bootstrap-server localhost:9092 --partitions 1 --replication-factor 1
bin/kafka-topics.sh --create --topic json-topic --bootstrap-server localhost:9092 --partitions 1 --replication-factor 1
bin/kafka-topics.sh --create --topic flume-mysql-topic --bootstrap-server localhost:9092 --partitions 1 --replication-factor 1
bin/kafka-topics.sh --create --topic flume-input-topic --bootstrap-server localhost:9092 --partitions 1 --replication-factor 1
```
