## 🚀 使用指南

### 快速启动

```bash
# 进入项目目录
cd /mnt/d/s/作业

# 启动所有服务
./start_all.sh

# 查看数据层
/usr/local/hadoop/bin/hdfs dfs -ls -R /warehouse/gmall/
```

### 执行 ETL 流程

```bash
# ODS 层数据加载
python3 ods_data_loader.py

# DIM 层数据处理
python3 dim_data_processor.py

# DWD 层数据处理
python3 dwd_data_processor.py

# DWS 层数据汇总
python3 dws_data_processor.py
```

### 查看数据

```bash
# 查看 ODS 层
/usr/local/hadoop/bin/hdfs dfs -cat /warehouse/gmall/ods/user_info

# 查看 DIM 层
/usr/local/hadoop/bin/hdfs dfs -cat /warehouse/gmall/dim/dim_user

# 查看 DWD 层
/usr/local/hadoop/bin/hdfs dfs -cat /warehouse/gmall/dwd/dwd_order_info

# 查看 DWS 层
/usr/local/hadoop/bin/hdfs dfs -cat /warehouse/gmall/dws/dws_gmv_stats
```

## ⚠️ 注意事项

### 1. 服务启动顺序
1. MySQL
2. Hadoop (NameNode → DataNode)

### 2. 端口占用
如果端口被占用，请先停止占用进程：
- MySQL: 3307
- Hadoop: 9000

### 3. 目录权限
确保 WSL 用户（jules）有权限访问 `/tmp/hadoop-jules/` 目录。

### 4. Java 版本
Hive 3.1.3 需要 Java 8 环境，已配置。
