### 8.1 前端开发

```bash
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

### 8.2 大数据服务

```bash
# 重启Kafka
cd /mnt/d/s/作业
chmod +x restart_kafka_full.sh
echo '54088Cnm,' | sudo -S ./restart_kafka_full.sh

# 启动HDFS
/usr/local/hadoop/bin/hdfs --daemon start namenode
/usr/local/hadoop/bin/hdfs --daemon start datanode

# 启动Flume
cd /opt/flume
./bin/flume-ng agent --conf conf --conf-file /mnt/d/s/作业/flume-conf/flume-mysql.conf --name a1
```

### 8.3 数据库

```bash
# 设置数据库
python3 setup_ecommerce_db.py

mysql -h localhost -P 3307 -u testuser -ptestpass
```

## 十二、注意事项

1. **中文路径问题**：Flume等Java程序无法处理中文路径，脚本需放在 `/tmp/workspace`
2. **权限问题**：Kafka、Hadoop目录需要当前用户所有权
3. **服务依赖**：Kafka依赖Zookeeper，Flume依赖Kafka和Hadoop
4. **端口冲突**：确保端口未被占用
